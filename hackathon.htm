Machine Learning Hackathon

Grade 16-18

Overview: Teams will create a model that recognizes specific objects from a set of images. Participants will gather images, train models, and test them on a hidden object in a final image. The goal is to design a model that accurately identifies the hidden object based on selected images.

Team Composition: Teams of 4: developer, data handler, researcher, presenter.

Event Timing: 9:30 AM - Introduction & Briefing, 9:45 AM - Image Training, 10:30 AM - Testing.

Rules: Use original work, cite resources, proper documentation is required. Submit models and supporting docs via the shared format.

Criteria Weightage
Model Accuracy 40%
Image Selection Quality 20%
Model Efficiency 15%
Presentation & Explanation 15%
Proper Documentation 10%

Sample Challenge: Train a model to detect a hidden object in an image using provided training images. The team with the most accurate model wins.

FAQs

A1: A platform for image uploading and model training will be provided at the event.

A2: Submission instructions will be shared during the event. Follow the guidelines for valid submissions.
